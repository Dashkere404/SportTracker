# Спортивный трекер
- Пользователь может добавлять и удалять упражнения, которые хотел бы выполнить в будущем. Также он может редактировать уже существующие упражнения.
- При выполнении пользователь может отметить это в приложении, нажав на выполненное упражнение.
- Название - обязательный параметр - String
- Группа мышц - выбирается из предложенного списка
- Вес - Double
- Число повторений в одном подходе - Int
- Количество подходов - Int
# Запуск
- Склонировать репозиторий на локальный компьютер в Android
- Запустить приложение на эмуляторе, либо на телефоне через провод
